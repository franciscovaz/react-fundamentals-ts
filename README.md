## 💻 Getting started

**Clone the project and access the folder**

```bash
$ git clone https://github.com/franciscovaz/react-fundamentals-ts.git
```

```bash
$ cd react-fundamentals-ts
```

**Follow the steps below**

```bash
# Install the dependencies
$ npm install

# Start the client
$ num run dev
```
